# Code of Conduct

我们承诺让所有人在 NexCore SDK 社区有一个友好、包容的体验,不论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 期望

- 对他人表现共情和善意
- 尊重不同观点和经验
- 以建设性方式给予和接受反馈
- 为错误承担责任、向受影响者致歉、从中吸取教训
- 关注社区整体利益,而不仅个人

## 不可接受的行为

- 性化语言或图像、性别歧视、人身攻击
- 蓄意挑衅、侮辱、贬损评论
- 公开/私下骚扰
- 未经明确同意发布他人隐私信息(地址、邮箱等)
- 其他职业场景中合理认为不当的行为

## 报告

通过 `conduct@nexcores.net` 邮件报告问题。所有投诉都会被审阅与调查,处理结果会及时回应。我们会保护报告人的隐私和安全。

## 归属

本《行为准则》改编自 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
